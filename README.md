# wabi-todo list and react starter kit for Wabi

1. create-react-app
=> 크리에이트 리엑트 앱으로 초기 세팅

2. yarn run eject / npm run eject
=> creat-react-app setting 을 푼다.

3. yarn add node-sass sass-loader less less-loader --save-dev / npm install node-sass sass-loader less less-loader --save-dev
=> sass 추가.

4. webpack.config.dev.js , webpack.config.prod.js
=> Sass, Less 컴파일링 되는 부분 추가
(https://velopert.com/3447) 참조.

5. yarn add classnames
빠른 className 삽입을 위해서 추가.
